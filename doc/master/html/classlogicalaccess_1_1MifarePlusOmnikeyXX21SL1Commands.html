uint16_t  keyslot 
)
overridevirtual

The implementation for XX21 needs to use the GENERIC SESSION provided by HID in order to send the AES Authenticate to the card.

This doesn't work under Linux, presumably because the driver doesn't support it.

Reimplemented from logicalaccess::MifarePlusSL1Commands.
